# VNC Viewer

A simple VNC viewer built with egui and miniquad.

It connects to a VNC server, displays the remote desktop, and sends basic input back to the server.

## Supports

- VNC connections by address and port
- Username and password credentials
- Saved connection details through the system keyring
- Remote desktop display
- Mouse movement, clicks, and scrolling
- Keyboard input
- Clipboard text sync
- F8 menu with disconnect, fullscreen, send F8, and send Ctrl+Alt+Del
- Shared VNC sessions
